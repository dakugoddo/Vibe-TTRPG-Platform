# Доработка Канваса: Excalidraw-like Рисование, Изображения, Порталы

Расширение `InfiniteCanvas` до полноценного инструмента рисования с векторными линиями, изображениями и переосмысленной навигацией по элементам.

## Реализовано

### Фаза 1 — Базовое рисование ✅
- Типы `DrawElement`, `StrokeStyle`, `LineCap` в `canvasTypes.ts`
- Zustand-стор `canvasDrawStore.ts`
- Панель инструментов `CanvasToolbar.tsx` (top-center, glassmorphism)
- Рисование Pen/Line/Rect/Ellipse в `InfiniteCanvas.tsx`
- Стили: цвет, толщина, dash-стиль, прозрачность
- Сохранение/загрузка через Yjs
- Удаление элементов (Delete/Backspace)
- Клавиатурные шорткаты (H/V/P/L/R/O/Escape)
- Middle-click pan для всех режимов
- Right-click НЕ рисует

### Фаза 2 — Стили и редактирование ✅
- Наконечники линий: none/arrow/circle/diamond/square (start + end)
- `LineCapDecoration` компонент с геометрией
- Drag-to-move выделенных элементов (snapshot pattern)
- `PointHandles` — перетаскиваемые точки на линиях/полилиниях
- `ResizeHandles` — угловые ресайз-ручки для прямоугольников/эллипсов/текста
- Контекстный стиль-панель при выделении → мгновенное изменение свойств

### Фаза 2.1 — Исправления и улучшения ✅
- **Баг 1:** Селекторы наконечников скрыты для фигур (rect/ellipse), видны только для линий
- **Баг 2:** Курсор `move` при наведении на выделяемые элементы в режиме select
- **Баг 3:** Сглаживание пера — min distance 8px + RDP-алгоритм + Konva `tension` для сплайнов
- **Баг 4:** Компактный 2-рядный стиль-панель: цвета 2×5, секции с подписями
- **Новое:** Заливка фигур — палитра полупрозрачных цветов + "без заливки"
- **Новое:** `getSelectedElementType()` для контекстного UI

### Фаза 3 — Мульти-выбор, Текст, Undo/Redo ✅
- **3.1 Marquee selection** — рамка выделения, elementsInRect с line-segment intersection
- **3.2 Undo/Redo** — undoStack/redoStack (max 50), Ctrl+Z/Ctrl+Shift+Z/Ctrl+Y
  - Все операции записывают историю: рисование, drag-move, point editing, resize, rotate, style change, delete, duplicate
- **3.3 Copy/Paste/Duplicate** — Ctrl+C/V/D, сдвиг +20px, новые ID, Ctrl+A select all
- **3.4 Текстовый инструмент** — textarea overlay, Konva `<Text>`, двойной клик → редактирование, шрифты/размеры/выравнивание
- **3.5 Выравнивание** — 6 кнопок (left/center/right/top/middle/bottom) при мульти-выборе ≥2
- **3.6 Поворот элементов** — RotationHandle для shapes/text, Stage pointer для точного угла
- **Доп:** Group bounding box — единая рамка вокруг мульти-выбора
- **Доп:** Resize handles для текстовых элементов

### Фаза 4 — Изображения, Порталы, Active Elements ✅
- **4.1 Image tool** — `I` шорткат, file dialog, drag-and-drop, `use-image` + Konva `<Image>`, resize/rotate/move, автомасштаб 600px, placeholder ⏳
- **4.2 Portal preview** — `PortalPulseRing` (анимация), swirl icon 🌀, target label, cosmic shadows
- **4.4 Active Elements** — рисунки в dropdown с иконками, клик→центрирование+выделение, draw count в badge
- **4.5 Extended fills** — 17 цветов (no-fill + 8 semi-transparent + 8 solid), 2-row layout

### Фаза 5 — Фреймы, Лассо, Overflow ✅
- **5.0 Фреймы** — `DrawElement type='frame'`, шорткат `F`, контейнер с лейблом, resize/rotate, prompt-имя, fill/stroke стили
- **5.3 Лассо** — свободная форма выделения, pointInPolygon ray-casting, визуализация полигона
- **5.4 Overflow-меню** — кнопка "⋯" с выпадающим dropdown, лассо в оверфлоу

### Фаза 6 — Багфиксы фреймов, Z-order, UX-улучшения ✅
- **6.1 Фрейм: drag за шапку** — перетаскивание только за header bar (fy-20..fy), клик на body не инициирует drag
- **6.2 Фрейм: перемещение дочерних элементов** — `getChildrenOfFrame()` определяет объекты внутри; при drag фрейма все дети двигаются вместе
- **6.3 Фрейм: inline-ввод имени** — заменён `window.prompt` на inline `<input>` overlay; авто-открывается после создания фрейма
- **6.4 Фрейм: двойной клик на label → переименование** — `handleFrameLabelDblClick` открывает inline-ввод
- **6.5 Фрейм: шапка наследует цвет** — headerFill вычисляется из fill/stroke элемента, а не фиксированный синий
- **6.6 Фрейм: стиль рамки (strokeStyle)** — solid/dashed/dotted теперь все работают для фреймов (не hardcoded dash)
- **6.7 Drag не отпускается на UI** — `isDraggingGlobal` flag в store + `pointer-events: none` на HudBar/Drawers/WindowManager во время drag
- **6.8 Лассо: корректное выделение** — проверка центра + 4 углов элемента через pointInPolygon; копия массива перед очисткой
- **6.9 Z-Order / Слои** — панель "Слои" в стиль-панели: На задний план / Назад / Вперёд / На передний план; `reorderElements()` утилита; рендеринг отсортирован по zIndex
- **6.10 Переименование** — "Активные элементы" → "Объекты на канвасе" в ru.json
- **6.11 Имена в Active Elements** — frameLabel для фреймов, objectName для фигур отображаются вместо типа
- **Новое:** `objectName`, `showName`, `description` поля в DrawElement

### Фаза 7 — Названия объектов, описания, линии ✅
- **7.1 Названия объектов** — `objectName` отображается centerX над фигурами (rect/ellipse); видимость через `showName`
- **7.2 Текст внутри фигур** — `description` рендерится внутри rect/ellipse с word-wrap; двойной клик на фигуру → textarea overlay для ввода
- **7.3 Линии: промежуточные точки** — двойной клик на линию → добавление точки в ближайший сегмент; point-on-segment projection для точного позиционирования
- **7.4 Метки линий** — `objectName` рендерится по центру линии с фоновой подложкой; Alt+двойной клик → ввод имени
- **7.5 Переименование объектов** — Alt+двойной клик на rect/ellipse/line → inline input для objectName
- **7.6 Toggle видимости имении** — кнопка глаз в стиль-панели при выделении; подсказка "Alt+2×клик → название"
- **7.7 Описания в Active Elements** — preview описания (📝) в списке объектов канваса

### Фаза 8 — Refinements, Форматирование и Smart Snaping ✅
- **8.1 Линии: исправления имён** — имя на линии теперь центрируется корректно даже при 2х точках, toggle видимости работает.
- **8.2 Текст внутри фигур: форматирование** — удален отдельный инструмент Text. Фигуры (Прямоугольник/Эллипс) теперь берут на себя эту роль (2×клик).
- **8.3 Типографика фигур** — в стиль-панели для выделенных фигур/линий доступны настройки: `Шрифт`, `Размер`, `Выравнивание`, `Цвет текста`, `Прозрачность текста`.
- **8.4 Независимая прозрачность** — разделение `opacity` на `fillOpacity` и `strokeOpacity`. В стиль-панели 3 раздельных слайдера: для обводки, для заливки и общего полупрозрачного вида текста.
- **8.5 Shift-snap вращения** — удержание `Shift` при вращении фигуры (через верхний rotation handle) примагничивает угол к шагам по 30°.
- **8.6 Shift-lock перемещения** — удержание `Shift` при перетаскивании (drag-move) лочит перемещение по доминирующей оси (строго по горизонтали или вертикали).
- **8.7 Smart Snap Guides** — при перетаскивании одного элемента центры и края автоматически выравниваются с центрами/краями других объектов (tolerance: 5 screen pixels), с отображением розовых вспомогательных линий 🩷.

## Фаза 4 — Изображения, Порталы, Active Elements ✅

### 4.1 Загрузка изображений ✅
- Инструмент `image` (шорткат `I`) → файловый диалог
- Drag-and-drop изображений с рабочего стола прямо на канвас
- `DrawElement type='image'` с `imageUrl` (data URL)
- Рендер через Konva `<Image>` + `use-image`
- Масштабирование (ResizeHandles), перемещение и поворот в select mode
- Автомасштаб: max 600px на любую сторону
- Заглушка пока загружается (⏳ placeholder)
- Стиль-панель скрывает обводку для image-only выборов

### 4.2 Превью порталов ✅
- Пульсирующее кольцо `PortalPulseRing` (requestAnimationFrame)
- Декоративный внутренний пунктирный круг
- Эмодзи-иконка 🌀 в центре
- Подпись имени портала + имя целевого канваса (→ TargetName)
- Тень и свечение для космического вида

### 4.3 Фреймы → Отложено в Фазу 5
- Сложная фича, требует отдельного DrawElement type='frame'
- Все объекты внутри фрейма перемещаются вместе с ним

### 4.4 Содержимое канваса (рефакторинг Active Elements) ✅
- Группированный список: **Рисунки** / Порталы / Токены / Закреп. окна
- Рисунки показываются с иконками по типу (✏️/◻️/🔤/🖼️)
- Клик на элемент → центрирование + выделение
- Текстовые элементы показывают превью текста
- Счётчик общего количества включает draw elements
- Скроллируемый dropdown с max-height: 60vh

### 4.5 Расширенная заливка ✅
- 17 цветов: no-fill + 8 полупрозрачных + 8 сплошных
- 2 ряда в палитре: прозрачные сверху, сплошные снизу
- Цвета: purple, red, orange, yellow, green, cyan, pink, white/gray

## Фаза 5 — Фреймы, Лассо, Overflow ✅

### 5.0 Фреймы ✅
- `DrawElement type='frame'` с `frameLabel`
- Инструмент Frame (шорткат `F`) — рисование как rect
- Prompt для имени фрейма при создании
- Лейбл сверху с полупрозрачным фоном
- Пунктирная рамка + полупрозрачная заливка
- Resize/Rotate handles, стиль-панель (fill/stroke)
- Отображается в Active Elements с иконкой 🔲

### 5.1 Встроенные web-страницы → Отложено
- `DrawElement type='iframe'`
- Синхронизация iframe между игроками (сложно, требует proxy)

### 5.2 Указка (мультиплеер) → Отложено
- Специальный инструмент-указка
- Требует real-time WebRTC/cursor sharing

### 5.3 Лассо-выделение ✅
- Свободная форма выделения полигоном
- Алгоритм pointInPolygon (ray-casting)
- Визуализация полигона при рисовании
- Авто-переключение в select mode
- Кнопка в overflow-меню

### 5.4 Overflow-меню (доп. инструменты) ✅
- Кнопка "⋯" в toolbar
- Выпадающий dropdown с редкими инструментами
- Лассо в overflow

10. **Undo/Redo:** Ctrl+Z отменяет drag/resize/rotate/style/delete

21. **Названия:** Alt+2×клик на фигуру/линию → inline input, имя отображается над объектом
22. **Описания:** 2×клик на rect/ellipse → textarea для описания, текст внутри фигуры
23. **Линии:** 2×клик на линию → добавление промежуточной точки

28. **Форматирование фигур:** Выделить фигуру → изменить шрифт, размер, цвет текста и прозрачность → описание должно обновиться.
29. **Раздельная прозрачность:** Изменить слайдер Заливки (изменяет только фон) и Обводки (изменяет только границу).
30. **Smart Snap Guides:** Перетаскивание объекта рядом с другим → появление розовых вспомогательных линий и примагничивание координат.
31. **Shift Snap/Lock:** Удержание Shift при повороте лочит шаг 30°, при перетаскивании лочит ось X или Y.
