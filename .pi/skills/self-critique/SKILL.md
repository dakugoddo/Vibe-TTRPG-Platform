# 🔍 Self-Critique & Reflection

> Когда применять: после КАЖДОГО слайса кода.

---

## 1. ЧЕКЛИСТ (пройти обязательно)

### Код
```
☐ TypeScript strict mode — нет any без нужды
☐ Нет проп-дриллинга > 2 уровней
☐ useMemo/useCallback на тяжёлых вычислениях
☐ useEffect с cleanup (подписки, таймеры, интервалы)
☐ Нет утечек памяти (Konva listeners, WebSocket)
☐ Обработаны ошибки (try/catch, error boundaries)
```

### UI
```
☐ Z-index по иерархии (0→10→20→30→40→50→9999)
☐ Стили через theme.ts (не хардкод)
☐ Контекстное меню по шаблону (секция 7 UI Architecture)
☐ Удаление через ConfirmDialog (не window.confirm)
☐ Drag & drop через DragDropPopover
☐ i18n для всех строк (t('key'))
```

### Canvas
```
☐ Координаты world vs screen корректны
☐ Throttle на drag/move/zoom
☐ Fog of war не сломан (новые слои не перекрывают)
☐ Pinned окна обновляются при zoom/pan
☐ Snap-to-grid работает с новыми элементами
```

### Мультиплеер
```
☐ Yjs синхронизация не сломана
☐ Права доступа: игрок ≠ ГМ
☐ Awareness (курсоры) не конфликтует
☐ FileSync debounce не нарушен
☐ Два клиента не создают конфликт
```

### Obsidian-совместимость
```
☐ .md формат не сломан (YAML frontmatter)
☐ [[wiki-ссылки]] работают
☐ Уникальность имён в general DB
```

### TTRPG
```
☐ useCalculatedStat context bubbling
☐ DiceParser edge cases (0d20, 100d100, -mod)
☐ Статы не NaN/undefined
☐ Инвентарь не дублируется
```

## 2. EDGE CASES (TTRPG-специфичные)

После каждой фичи — проверь эти сценарии:

```typescript
// 1. Статы
- Стат не найден на entity → bubble к parent
- Стат = 0 (минимальное значение)
- Стат отрицательный (редко, но бывает — низкий INT)
- Стат > 30 (божественный уровень, легендарные предметы)

// 2. Дайсы
- 1d1 (минимальный бросок)
- 100d100 (максимальный)
- 1d20-10 (отрицательный итог)
- 1d20+$nonexistent (переменная не найдена)

// 3. Canvas
- Токен за границами fog of war
- 100 токенов на одном канвасе (перформанс)
- Drag токена за границы экрана
- Переключение канваса во время drag

// 4. Мультиплеер
- Два игрока двигают один токен
- Игрок отключился во время боя
- ГМ удалил сущность, которую игрок редактирует
- FileSync конфликт: Obsidian + Yjs одновременно

// 5. Инвентарь
- Drag предмета из general в user (копия или перемещение?)
- Предмет с родителем (вложенные атаки) — переносить рекурсивно?
- Два игрока пытаются взять один предмет
```
